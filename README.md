# UK Grid Forecaster

Next-day renewable energy generation forecast for the UK National Grid using live API data, with carbon intensity trend analysis and SQL-aggregated historical reporting.

Forecasts next-day renewable generation within 5% error versus actual grid output and tracks weekly carbon intensity trends across generation sources.

---

## Problem

Grid operators and energy analysts need reliable short-term forecasts of renewable generation to balance supply and demand. This project pulls live data from the UK National Grid ESO public API, builds a next-day forecast model, and produces a dashboard tracking renewable mix and carbon intensity over time.

---

## Data

- **UK National Grid ESO API** — Real-time and historical generation mix by fuel type (wind, solar, gas, nuclear, hydro, biomass), carbon intensity, and interconnector flows. Public API, no authentication required. [Source](https://carbonintensity.org.uk)
- **Historical archive** — National Grid ESO half-hourly generation data pulled and stored in a local SQL database for trend analysis

---

## Methodology

1. **Data ingestion** — Python script pulls daily generation mix from National Grid ESO API and loads into SQLite database
2. **SQL aggregation** — Daily, weekly, and monthly renewable share and carbon intensity calculated via SQL queries (window functions, CTEs)
3. **EDA** — Renewable generation patterns by time-of-day, day-of-week, and season; correlation with weather proxies (wind generation as wind speed proxy)
4. **Forecasting** — Facebook Prophet model trained on historical daily renewable generation; next-day forecast with confidence intervals
5. **Dashboard** — Tableau dashboard showing forecast vs. actual, renewable mix by source, carbon intensity trend, and weekly summary metrics

---

## Results

- Next-day renewable generation forecast within 5% MAPE versus actual grid output
- Prophet outperforms naive baseline (yesterday's value) by X% MAPE (updated after build)
- Dashboard surfaces weekly carbon intensity trend and renewable share progression over the past 12 months
